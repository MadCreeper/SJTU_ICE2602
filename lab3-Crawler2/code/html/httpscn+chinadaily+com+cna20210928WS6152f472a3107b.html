观中国 | 印度学者谈阿富汗“变天”：谁还认为美国靠得住，谁就是自欺欺人

作者： 苏丁德拉·库尔卡尼 来源：中国日报网
2021-09-28 18:53　
 
 
 
分享
分享到

导读

美国近期从阿富汗仓促撤军，再次证明了“国霸必衰”的历史规律，也让美国在全世界面前信誉大跌。阿富汗的和平稳定重建，是国际社会，尤其是地区国家的共同责任，也和“一带一路”倡议的目标相契合。但作为地区重要国家，印度在阿富汗问题上立场相对孤立，中印、印巴协调不足，阻碍了地区合作。

 

任何一个大国，如果仅凭借其先天的地缘政治优势，就频频对远近各国发动战争，那么它势必会削弱自己，甚至可能自掘坟墓。纵观人类历史，此类案例比比皆是。

自二战结束以来，美国的对外军事行动就没停过，不是在这国打仗就是在那国打仗。近年来，美国接连出兵阿富汗和伊拉克——显然，它并没有从先前越南战争的失败中吸取教训。

然而美国最终也没能跳出历史规律：上个月，它被迫结束了长达20年的阿富汗战争，这场毫无意义的战争不可避免地让它付出了巨大代价。

 

图片来源：中国日报

在阿富汗战争背后，有两个事实令人震惊：第一，美国在这场所谓的“反恐战争”中造成了数万阿富汗人死亡，数千名美国和盟国的军人以及平民丧生。第二，美国政府在这场战争上花费了超过2万亿美元，而与此同时数百万普通美国民众正饱受日益加剧的不平等和贫困之苦。

塔利班夺取胜利后，美国的国际声誉遭受重创。因此，不论在亚洲还是世界，谁要是还认为美国可以成为可靠的安全伙伴，谁就是在自欺欺人。

美国的外交政策有一个根本性的问题：在它眼中，所有国家都是“主权国家”，但各国的主权似乎有高有低，因此美国自认为有权干涉其他国家的内部事务。

不知美国领导人什么时候才能明白，在一个要求世界秩序彻底民主化的时代，这种外交政策毫无作用，只会适得其反。

图片来源：中国日报

既然阿富汗已经恢复了主权，它最大、最紧迫的需求就是和平、稳定与民族和解，并立即重建其饱受战争蹂躏的国民经济。这不仅是塔利班等阿富汗社会政治力量的共同责任，也是国际社会，特别是地区国家的共同责任。这需要印度、中国、巴基斯坦、俄罗斯、伊朗等周边国密切磋商，相互协调，形成解决各方合理关切的共同战略。

双边对话，以及包括上海合作组织和“金砖国家”等在内的多边机制，正在积极推动相关进展，这无疑是个好兆头。

该地区所有国家，以及包括美国和欧洲在内的其他国际社会的主要共同关切是：阿富汗不能再为那些在别国煽动恐怖主义、极端主义或分裂主义的组织提供避难所。塔利班高级官员在这方面作出了承诺，并表示希望与所有国家建立友好关系。这一表态值得赞赏，表明塔利班已经认识到了自身在1996-2001年执政时所犯的错误，不能重蹈覆辙。

图片来源：中国日报

塔利班现在的首要任务是向阿富汗人民和世界证明，他们有能力治理好自己的国家，重建经济，弥补民生短板。

为阿富汗提供急需的人道主义援助，促进投资贸易，完善现代化基础设施，改善内外部互联互通，国际社会义不容辞。这正是中国“一带一路”倡议的目标所在。因此塔利班称中国为“朋友”，并表示阿富汗愿意加入“一带一路”倡议，这毫不奇怪。

诚然，塔利班的社会宗教思想和行为方式仍有一些值得怀疑的方面，特别是在妇女权利方面。但与此同时，西方国家不应将自己的“现代”标准强加给受传统束缚的阿富汗社会。社会改革是一个漫长的过程，必须是由内而外自发产生的。

图片来源：中国日报

尽管地区国家正在凝聚一种广泛的共识，但是这其中依然存在两个难题。

第一，中印两国在阿富汗问题上沟通、协调与合作不足。这显然是由于印度方面缺乏信心所致。

印度总理莫迪所领导的政府犯了一个错误：他们把所有外交“鸡蛋”都放在了美国“篮子”里。后果则是，印度政府在阿富汗问题上面临相对孤立的局面。印方应该纠正这个错误。具体来说，在帮助阿富汗和平、稳定复苏方面，印度应该与中国共同努力，因为中印两国在这方面的关切和利益是相一致的。

第二，在阿富汗问题上，印度和巴基斯坦到目前为止完全没有对话，两国之间的成见和矛盾很深，没有相互合作，只有相互指责。

其结果是，南亚区域合作联盟（阿富汗是成员国之一）近乎停摆。印度和巴基斯坦都应该认识到，阿富汗已经开启了一个充满希望的新时代。这对印巴两国来说既创造了一个机会，也意味着新的责任，要求两国放下历史矛盾，为南亚和平与共同繁荣的新篇章而努力，最终造福17亿南亚人民。
