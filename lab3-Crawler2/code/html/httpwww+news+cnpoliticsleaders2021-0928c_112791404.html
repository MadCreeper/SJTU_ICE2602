　　新华社北京9月28日电 今年是中国共产党成立100周年。9月30日是国家设立的烈士纪念日，当天上午，习近平等党和国家领导人将同各界代表一起，在天安门广场向人民英雄敬献花篮。

　　届时，中央广播电视总台将进行现场直播。
